	====================

to provide you with the source code of those components: this

  o linux.tgz
    The Linux kernel source code (based on Linux 2.4, plus the ARM and
    StrongARM patches)

  o distro.tgz
    Basic Linux utilities, such as bash, which are installed on your
    player (based on the ARM Debian distribution)

  o dosfstools.tgz
    Tools to make and check FAT filesystems

  o modutilssrc.tgz
    Tools for linux modular device drivers
